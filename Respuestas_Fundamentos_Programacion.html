Escenario 1: Sistema de Descuentos en Tienda de Abarrotes

Análisis y Determinación de Requerimientos

El problema requiere desarrollar una aplicación que calcule el desglose del pago de un cliente aplicando descuentos según la cantidad de unidades compradas.

Requerimientos Funcionales:

  • Recibir cantidad de unidades (número entero positivo)
  • Recibir precio unitario del producto
  • Calcular subtotal (cantidad × precio)
  • Aplicar descuento según reglas establecidas
  • Mostrar desglose: subtotal, descuento, total a pagar

Reglas de Negocio:

  • Más de 100 unidades: 40% de descuento
  • Entre 25 y 100 unidades: 20% de descuento
  • Entre 10 y 24 unidades: 10% de descuento
  • Menos de 10 unidades: Sin descuento

Variables Utilizadas

Variable Tipo Descripción Ejemplo
cantidad entero Número de unidades a comprar 50
precio decimal Precio unitario del producto 15.50
subtotal decimal Cantidad × precio 775.00
porcentajeDescuento decimal Porcentaje a aplicar (0.0 a 0.4) 0.20
descuento decimal Monto del descuento 155.00
total decimal Subtotal - descuento 620.00

Proceso de Desarrollo

El desarrollo siguió una metodología estructurada:

  1. Análisis del problema: Identificación de entradas, procesos y salidas
  2. Diseño de la lógica: Estructuras condicionales anidadas para aplicar descuentos
  3. Implementación: Codificación usando JavaScript con validaciones
  4. Pruebas: Casos de prueba para cada rango de descuento

Instrucciones Condicionales

Se utilizó una estructura if-else if-else para implementar las reglas de descuento:

if (cantidad > 100) { porcentajeDescuento = 0.40; // 40% } else if (cantidad >= 25) { porcentajeDescuento = 0.20; // 20% } else if (cantidad >= 10) { porcentajeDescuento = 0.10; // 10% } else { porcentajeDescuento = 0; // Sin descuento }

Código Fuente Completo

class Descuento { calcular(cantidad, precio) { // Validación de entrada if (!Number.isInteger(cantidad) || cantidad <= 0) { throw new Error('La cantidad debe ser un número entero positivo'); } if (typeof precio !== 'number' || precio <= 0) { throw new Error('El precio debe ser un número positivo'); } // Cálculo del subtotal const subtotal = cantidad * precio; let porcentajeDescuento = 0; // Aplicar reglas de descuento if (cantidad > 100) { porcentajeDescuento = 0.40; // 40% } else if (cantidad >= 25) { porcentajeDescuento = 0.20; // 20% } else if (cantidad >= 10) { porcentajeDescuento = 0.10; // 10% } // Si cantidad < 10, no hay descuento (0%) // Cálculos finales const descuento = subtotal * porcentajeDescuento; const total = subtotal - descuento; return { subtotal: Math.round(subtotal * 100) / 100, descuento: Math.round(descuento * 100) / 100, total: Math.round(total * 100) / 100, porcentaje: porcentajeDescuento * 100 }; } } // Función para la interfaz web function calcularDescuento() { const cantidad = parseInt(document.getElementById('cantidad').value); const precio = parseFloat(document.getElementById('precio').value); try { const descuento = new Descuento(); const resultado = descuento.calcular(cantidad, precio); document.getElementById('resultado').innerHTML = `

Desglose del Pago:

Subtotal: $${resultado.subtotal.toFixed(2)}

Descuento (${resultado.porcentaje}%): $${resultado.descuento.toFixed(2)}

Total a pagar: $${resultado.total.toFixed(2)}

`; } catch (error) { document.getElementById('resultado').innerHTML = `

Error: ${error.message}

`; } }

Escenario 2: Calculadora de Días del Mes

Análisis y Determinación de Requerimientos

El problema requiere una aplicación que determine la cantidad de días de un mes específico considerando años bisiestos.

Requerimientos Funcionales:

  • Recibir número del mes (1-12)
  • Recibir año (número entero positivo)
  • Determinar si el año es bisiesto
  • Calcular días del mes según reglas establecidas
  • Mostrar resultado con nombre del mes y cantidad de días

Reglas de Negocio:

  • Meses con 31 días: Enero, Marzo, Mayo, Julio, Agosto, Octubre, Diciembre
  • Meses con 30 días: Abril, Junio, Septiembre, Noviembre
  • Febrero: 28 días (29 en año bisiesto)
  • Año bisiesto: múltiplo de 4 y no de 100, O múltiplo de 400

Variables Utilizadas

Variable Tipo Descripción Ejemplo
mes entero Número del mes (1-12) 2
anio entero Año a evaluar 2024
esBisiesto booleano Indica si el año es bisiesto true
dias entero Número de días del mes 29
nombreMes cadena Nombre del mes "Febrero"

Lógica de Años Bisiestos

La determinación de años bisiestos sigue una lógica específica implementada con operadores lógicos:

const esBisiesto = (anio % 4 === 0 && anio % 100 !== 0) || (anio % 400 === 0);

Esta expresión evalúa:

  • Condición 1: Es múltiplo de 4 Y no es múltiplo de 100
  • Condición 2: Es múltiplo de 400
  • Resultado: Si cualquiera de las dos condiciones es verdadera, es bisiesto

Código Fuente Completo

class Fecha { obtenerDiasMes(mes, anio) { // Validación de entrada if (!Number.isInteger(mes) || mes < 1 || mes > 12) { throw new Error('El mes debe ser un número entero entre 1 y 12'); } if (!Number.isInteger(anio) || anio <= 0) { throw new Error('El año debe ser un número entero positivo'); } // Determinar si es año bisiesto const esBisiesto = this.esAnioBisiesto(anio); let dias; // Determinar días según el mes if ([1, 3, 5, 7, 8, 10, 12].includes(mes)) { dias = 31; } else if ([4, 6, 9, 11].includes(mes)) { dias = 30; } else if (mes === 2) { dias = esBisiesto ? 29 : 28; } // Array con nombres de meses const nombresMeses = [ '', 'Enero', 'Febrero', 'Marzo', 'Abril', 'Mayo', 'Junio', 'Julio', 'Agosto', 'Septiembre', 'Octubre', 'Noviembre', 'Diciembre' ]; return { dias: dias, esBisiesto: esBisiesto, nombreMes: nombresMeses[mes], anio: anio }; } esAnioBisiesto(anio) { // Un año es bisiesto si: // - Es múltiplo de 4 Y no es múltiplo de 100, O // - Es múltiplo de 400 return (anio % 4 === 0 && anio % 100 !== 0) || (anio % 400 === 0); } } // Función para la interfaz web function calcularDiasMes() { const mes = parseInt(document.getElementById('mes').value); const anio = parseInt(document.getElementById('anio').value); try { const fecha = new Fecha(); const resultado = fecha.obtenerDiasMes(mes, anio); document.getElementById('resultado').innerHTML = `

Resultado:

${resultado.nombreMes} de ${resultado.anio} tiene ${resultado.dias} días

${mes === 2 ? `

${resultado.esBisiesto ? '(Año bisiesto)' : '(Año normal)'}

` : ''} `; } catch (error) { document.getElementById('resultado').innerHTML = `

Error: ${error.message}

`; } }

  • Determinar cuál es el mayor
  • Mostrar el resultado en pantalla
  • Manejar casos de empate (números iguales)

Casos Especiales a Considerar:

  • Los tres números son iguales
  • Dos números son iguales y mayores que el tercero
  • Validación de entrada para números positivos

Algoritmo de Comparación

Se utiliza un algoritmo secuencial que compara los números de forma ordenada:

let mayor = num1; // Asumir que el primero es el mayor if (num2 > mayor) { mayor = num2; } if (num3 > mayor) { mayor = num3; }

Código Fuente Completo

Escenario 4: Generador de Listas (Rango, Pares e Impares)

Análisis y Determinación de Requerimientos

El problema requiere generar tres listas diferentes a partir de dos números: una ascendente completa, una descendente de pares y otra descendente de impares.

Requerimientos Funcionales:

  • Recibir dos números enteros positivos
  • Generar lista ascendente con todos los números del rango
  • Generar lista descendente con números pares del rango
  • Generar lista descendente con números impares del rango
  • Mostrar las tres listas en pantalla

Variables Utilizadas

Variable Tipo Descripción Ejemplo
a, b entero Números de entrada 3, 8
menor, mayor entero Límites del rango ordenados 3, 8
ascendente array Lista de menor a mayor [3,4,5,6,7,8]
pares array Números pares descendente [8,6,4]
impares array Números impares descendente [7,5,3]

Estructuras de Repetición

Se utilizan tres bucles for diferentes para generar cada lista:

// Lista ascendente completa for (let i = menor; i <= mayor; i++) { ascendente.push(i); } // Lista descendente de pares for (let i = mayor; i >= menor; i--) { if (i % 2 === 0) pares.push(i); } // Lista descendente de impares for (let i = mayor; i >= menor; i--) { if (i % 2 !== 0) impares.push(i); }

Código Fuente Completo

class Rango { generarListas(a, b) { // Determinar menor y mayor const menor = Math.min(a, b); const mayor = Math.max(a, b); // Lista ascendente completa const ascendente = []; for (let i = menor; i <= mayor; i++) { ascendente.push(i); } // Lista descendente de pares const pares = []; for (let i = mayor; i >= menor; i--) { if (i % 2 === 0) pares.push(i); } // Lista descendente de impares const impares = []; for (let i = mayor; i >= menor; i--) { if (i % 2 !== 0) impares.push(i); } return { ascendente: ascendente, pares: pares, impares: impares }; } } // Función para la interfaz web function generarListas() { const a = parseInt(document.getElementById('numeroA').value); const b = parseInt(document.getElementById('numeroB').value); const rango = new Rango(); const resultado = rango.generarListas(a, b); document.getElementById('resultado').innerHTML = `

Listas Generadas:

Lista Ascendente: ${resultado.ascendente.join(', ')}

Pares Descendente: ${resultado.pares.join(', ')}

Impares Descendente: ${resultado.impares.join(', ')}

`; }

Escenario 5: Calculadora de Factorial

Análisis y Determinación de Requerimientos

El problema requiere calcular el factorial de un número entero positivo usando la fórmula n! = n × (n-1) × (n-2) × ... × 1.

Requerimientos Funcionales:

  • Recibir un número entero positivo
  • Calcular el factorial usando iteración
  • Mostrar el resultado en pantalla
  • Manejar casos especiales (0! = 1)

Fórmula Matemática:

  • n! = n × (n-1) × (n-2) × ... × 1
  • 0! = 1 (por definición)
  • Ejemplo: 5! = 5 × 4 × 3 × 2 × 1 = 120

Variables Utilizadas

Variable Tipo Descripción Ejemplo
n entero Número del cual calcular factorial 5
resultado entero Acumulador para el factorial 120
i entero Variable de control del bucle 1, 2, 3, 4, 5

Algoritmo Iterativo

Se utiliza un bucle for para calcular el factorial de forma iterativa:

let resultado = 1; for (let i = 1; i <= n; i++) { resultado *= i; }

Código Fuente Completo

class Factorial { calcular(n) { // Validación de entrada if (!Number.isInteger(n) || n < 0) { throw new Error('El número debe ser un entero no negativo'); } // Caso especial: 0! = 1 if (n === 0) return 1; // Cálculo iterativo del factorial let resultado = 1; for (let i = 1; i <= n; i++) { resultado *= i; } return resultado; } } // Función para la interfaz web function calcularFactorial() { const n = parseInt(document.getElementById('numero').value); try { const factorial = new Factorial(); const resultado = factorial.calcular(n); document.getElementById('resultado').innerHTML = `

Resultado:

${n}! = ${resultado.toLocaleString()}

Cálculo: ${n}! = ${n} × ${n-1} × ... × 1

`; } catch (error) { document.getElementById('resultado').innerHTML = `

Error: ${error.message}

`; } }

Escenario 6: Máximo Común Divisor (MCD) y Mínimo Común Múltiplo (MCM)

Análisis y Determinación de Requerimientos

El problema requiere calcular el MCD y MCM de dos números enteros positivos usando el algoritmo de Euclides.

Requerimientos Funcionales:

  • Recibir dos números enteros positivos
  • Calcular el MCD usando el algoritmo de Euclides
  • Calcular el MCM usando la relación: MCM = (a × b) / MCD
  • Mostrar ambos resultados en pantalla

Conceptos Matemáticos:

  • MCD: Mayor número que divide exactamente a ambos números
  • MCM: Menor número que es múltiplo de ambos números
  • Relación: MCD(a,b) × MCM(a,b) = a × b

Variables Utilizadas

Variable Tipo Descripción Ejemplo
a, b entero Números de entrada 12, 18
mcd entero Máximo Común Divisor 6
mcm entero Mínimo Común Múltiplo 36

Algoritmo de Euclides

Se utiliza el algoritmo de Euclides recursivo para calcular el MCD:

const mcd = (x, y) => y === 0 ? x : mcd(y, x % y); const mcm = (a * b) / mcd;

Código Fuente Completo

class McdMcm { calcular(a, b) { // Validación de entrada if (!Number.isInteger(a) || !Number.isInteger(b) || a <= 0 || b <= 0) { throw new Error('Ambos números deben ser enteros positivos'); } // Algoritmo de Euclides para MCD const mcd = (x, y) => y === 0 ? x : mcd(y, x % y); const resultadoMCD = mcd(a, b); const resultadoMCM = (a * b) / resultadoMCD; return { mcd: resultadoMCD, mcm: resultadoMCM }; } } // Función para la interfaz web function calcularMcdMcm() { const a = parseInt(document.getElementById('numeroA').value); const b = parseInt(document.getElementById('numeroB').value); try { const mcdmcm = new McdMcm(); const resultado = mcdmcm.calcular(a, b); document.getElementById('resultado').innerHTML = `

Resultado:

Números: ${a} y ${b}

MCD: ${resultado.mcd}

MCM: ${resultado.mcm}

Verificación: ${resultado.mcd} × ${resultado.mcm} = ${resultado.mcd * resultado.mcm}

Original: ${a} × ${b} = ${a * b}

`; } catch (error) { document.getElementById('resultado').innerHTML = `

Error: ${error.message}

`; } }

Escenario 7: Generador de Números Primos

Análisis y Determinación de Requerimientos

El problema requiere generar una lista de todos los números primos desde 1 hasta un número límite especificado.

Requerimientos Funcionales:

  • Recibir un número entero positivo (límite superior)
  • Verificar cada número para determinar si es primo
  • Generar lista con todos los números primos encontrados
  • Mostrar la lista en pantalla

Definición de Número Primo:

  • Un número primo es divisible únicamente por 1 y por sí mismo
  • El número 1 no se considera primo
  • El número 2 es el único primo par

Variables Utilizadas

Variable Tipo Descripción Ejemplo
n entero Número límite superior 20
lista array Array con números primos [2,3,5,7,11,13,17,19]
esPrimo función Función para verificar si es primo esPrimo(7) → true

Algoritmo de Verificación de Primos

Se utiliza un algoritmo optimizado que verifica divisibilidad hasta la raíz cuadrada:

const esPrimo = num => { if (num < 2) return false; for (let i = 2; i <= Math.sqrt(num); i++) { if (num % i === 0) return false; } return true; };

Código Fuente Completo

class Primos { calcular(n) { // Validación de entrada if (!Number.isInteger(n) || n <= 0) { throw new Error('El número debe ser un entero positivo'); } // Función para verificar si un número es primo const esPrimo = num => { if (num < 2) return false; for (let i = 2; i <= Math.sqrt(num); i++) { if (num % i === 0) return false; } return true; }; // Generar lista de números primos const lista = []; for (let i = 1; i <= n; i++) { if (esPrimo(i)) lista.push(i); } return lista; } } // Función para la interfaz web function generarPrimos() { const n = parseInt(document.getElementById('numero').value); try { const primos = new Primos(); const resultado = primos.calcular(n); document.getElementById('resultado').innerHTML = `

Resultado:

Números primos del 1 al ${n}:

${resultado.join(', ')}

Total encontrados: ${resultado.length} números primos

`; } catch (error) { document.getElementById('resultado').innerHTML = `

Error: ${error.message}

`; } }
